🚀 Как настроить Happ

  1. Скопируйте сгенерированную выше ссылку.
  2. В приложении Happ: перейдите во вкладку Subscriptions (Подписки) → нажмите Add (+) и вставьте ссылку.
  3. Нажмите кнопку Refresh (🔄) для загрузки серверов.
  4. Перейдите в Settings → Routing → выберите режим "Только заблокированные" (CIDR режим для экономии трафика и максимальной скорости).

🛠 Как создать свой авто-объединитель бесплатно

  1. Сделайте Fork репозитория на GitHub (или создайте новый репозиторий).
  2. Перенесите туда файлы `merge.py`, `.github/workflows/update.yml`, `index.html`, `styles.css` и `app.js`.
  3. В настройках репозитория: Settings → Actions → General → в самом низу в Workflow permissions выберите "Read and write permissions".
  4. Перейдите во вкладку Settings → Pages → в разделе Build and deployment выберите ветку main (папка / (root)).
  5. GitHub Actions будет автоматически обновлять ваши конфиги каждые 6 часов, а сайт будет доступен бесплатно.
