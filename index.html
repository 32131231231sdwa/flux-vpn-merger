🚀 Как добавить в Happ

  1. Способ 1 — по ссылке (для «Найти лучшую подписку»):
    Скопируйте ссылку → Happ → Подписки → + → «Добавить подписку по ссылке» → Вставить → Сохранить.
  2. Способ 2 — из буфера (для «Своя подписка»):
    Скопируйте серверы → Happ → Подписки → + → «Импорт из буфера обмена».
  3. Нажмите Refresh (🔄) для обновления серверов.
  4. Перейдите в Settings → Routing → выберите режим "Только заблокированные" (CIDR режим для максимальной скорости).
